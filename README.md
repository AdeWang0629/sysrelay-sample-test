## Development Environment

node 18.15.0v

## Run Locally  

Clone the project  

~~~bash  
  git clone https://github.com/AdeWang0629/sysrelay-sample-test.git
~~~

Go to the project directory  

~~~bash  
  cd sysrelay-sample-test
~~~

Install dependencies  

~~~bash  
npm install
~~~

Start the server  

~~~bash  
npm run start
~~~
